Updates on Are.na Canvas

Updated the UI

Blocks are resizable

Send values on connect

Nodes now send values on connection.

The fact that they send the value on connection. Making a connection can be like a button press.

Save values to a node

And then use that node as a bookmark for a location

Keypress to send value

Added this node that will hold a value in itself and then release and send the value forward on a keypress. So location bookmarks can be saved to a keypress.

OPEN IN OBSIDIAN

And the most rewarding part of using this open file format is that I can export and open this canvas directly in Obsidian. Make some edits in Obsidian and open it back into this Are.na Canvas. Which I think is so fucking cool!

I theory this would also work for all the other apps that import and export to this format. will have to test it out though.
